# Exercise
Given 2 arrays merge the content of each array 3 items
from each array.

**Ej.:**

**Input:**
```java
int [] firstArray = {1,2,3,4,5,6,7,8,9};
int [] secondArray = {10,11,12,13,14,15,16,17,18};
```

**Output: 0,100,200**
```java
int [] result = {1,2,3,10,11,12,13,4,5,6,13,14,15,7,8,9,16,17,18,19}
```
